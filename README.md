## 项目说明
> 电影智能推荐系统采用典型的MVC架构。系统使用Java 语言作为后台编程语言，提供RESTFUL接口，使用MYSQL作为数据库；智能推荐算法采用协同过滤算法，提供的推荐功能包括：视频推荐、好友推荐、圈子推荐等。项目的前端使用Vue2开发，前端项目地址为：[https://github.com/firery/mirs-web-vue](https://github.com/firery/mirs-web-vue)

# 安装方法

```
1. 定位到src/main/resources/properties/文件夹下
2. jdbc-example.properties --> jdbc-prod.properties 根据提示配置
3. mail-example.properties --> mail-prod.properties 根据提示配置
```
